# もちょVimプラグインだよ〜(●・▽・●)

## 使い方(o・∇・o)
### コマンド
```
 :Mocho
```

って打つとカレント行に  
(●・▽・●)  
って挿入されるみたいだよ〜(*ﾟ0ﾟ)わーお

### その他
.vimrcに

```
set statusline=%{g:StatusMocho()}
```

って書き込むと(●・▽・●)がステータスラインでクルクル回るようになるよ〜

## インストール(-^〇^-)
NeoBundleInstallすればいいみたいですよ〜(o・∇・o)
```
vim ~/.vimrc
```
でvimrcファイルを開いて
```
NeoBundle 'https://github.com/mizukmb/mocho.git'
```
って書き込んで
```
:NeoBundleInstall
```
でインストール完了だよ〜(o･∇･o)
